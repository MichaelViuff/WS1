# CLI Basics Course

Welcome to the **CLI Basics Course**! This course will introduce you to essential command-line interface (CLI) commands and concepts.

## Course Overview

- Introduction to the CLI
- Navigating the file system
- Managing files and directories
- Viewing and editing files
- Useful CLI tools and shortcuts

## Getting Started

1. Open your terminal or command prompt.
2. Follow along with each lesson and try the commands yourself.

## Example Commands

- `ls` / `dir` — List files and directories
- `cd` — Change directory
- `mkdir` — Create a new directory
- `touch` / `type nul >` — Create a new file
- `rm` / `del` — Remove files
